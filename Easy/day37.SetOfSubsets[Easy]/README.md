The *power set* of a *set* is the _set of all its subsets_. Write a function that, given a _set_, generates its _power set_.

For example, given the set `{1, 2, 3}`, it should return
```python
{
    {},
    {1},
    {2},
    {3},
    {1, 2},
    {1, 3},
    {2, 3},
    {1, 2, 3}
}
```
